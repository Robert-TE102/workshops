what is a local host?
A local Host is a host that refers to the current computer/client via an ip address.

What are nameservers, why are they needed?
A named server is a dedicated domain name e.g wikipidea.com, and they are needed to directly navigate to the website you are looking for instead of using an IP address.

Describe client and server: making a request, receiving a response
A client is the host IP address that sends packages to a dedicated server to make a request, to load up a website that you are trying to access, and it responds by sending packages back to the client, saying it's received the request and loads the website.
